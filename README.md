Online Internship Portal
Built a role-based web platform for students and companies to manage internship applications, including profile creation, internship posting, application tracking, and candidate selection.

Features
User registration and login with session management
Student module for profile creation, internship search, application, and tracking
Company module for posting internships, managing listings, and selecting candidates
Real-time application status updates
Dynamic interaction using AJAX

Tech Stack
Frontend: HTML, CSS, DHTML, JavaScript
Backend: Java (Servlets & JSP), PHP
Database: MySQL
Server: Apache Tomcat# Online-Internship-Portal
